Connect to Firebase

One-time setup to sync data across all devices

1
Go to console.firebase.google.com → Create a project (e.g. vasant-lawns) → Disable Google Analytics → Create
2
In your project → Build → Firestore Database → Create database → choose Start in test mode → Next → Enable
3
Go to Project Settings (gear icon) → Your apps → click </> Web → Register app → Copy the firebaseConfig values below

Your config is saved in this browser. Share the HTML file with the config filled in for others.
